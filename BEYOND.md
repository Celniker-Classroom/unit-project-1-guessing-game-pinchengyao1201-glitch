I added two extra features beyond the required guessing game:

1. Win streak tracking
- It is line 7, 27-28, 81-83, 119 in script.js. The streak increases whenever the player guess the number right but resets when the player gives up. This motivates the player to keep their streaks and not give up and press the give up button. It makes the game harder to get a high streak which makes it more fun. 

2. Keyboard support with Enter-to-guess
- It is in line 134-138 using an event listener to trigger the guess button when player presses Enter. This improves the game by making it more convenient and allow them to get a faster time. 

3. Sound Effect activated when Winning 
- It is included in line 29, 76-77 using an added mp3 recording, it plays when a player gets the right number. This makes the player be more satisfied after winning. It also makes the game less awkward and have some sort of music to it.
